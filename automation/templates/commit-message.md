# Template — Commit message

**Only commit when the maintainer explicitly asks** (`../../directives/github_workflow.md`). Never
push or open/merge a PR unless asked in the moment.

The repo's history uses short, imperative, capitalized summaries. Match that style:

```
<Imperative summary of what changed, ≤ ~70 chars>

<Optional body: why the change, anything load-bearing to note.>
```

Examples that match existing history:

```
Update partner inquiry Google Form link
```

```
Add editing map and automation start guide
```

Keep it factual and scoped to the diff. Do not describe changes you didn't make. Do not add a
push/deploy step — stop after committing.
