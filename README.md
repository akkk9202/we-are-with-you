## Deploy to GitHub Pages

1. Create a new GitHub repository (e.g. `wearewithyou`)
2. Upload all files maintaining this folder structure:
   ```
   /
   ├── index.html
   ├── we-are-with-you.html
   ├── beat-and-breeze.html
   ├── taps-of-love.html
   ├── winds-of-love.html
   ├── nado-passport.html
   ├── gyco.html
   ├── media.html
   ├── join.html
   ├── contact.html
   ├── css/
   │   └── style.css
   └── js/
       └── main.js
   ```
3. Go to **Settings → Pages → Source → main branch / root**
4. Your site will be live at `https://[username].github.io/[repo-name]/`

## Before Publishing

- Replace `[Insert official email]` in `contact.html` with real email
- Replace `[Insert Instagram link]` and `[Insert YouTube link]` with real URLs
- Add real photos by replacing the placeholder `<div class="two-col__image-placeholder">` elements with `<img>` tags
- Update the hero background image URL in `index.html` if desired

## Design Tokens

- **Navy** `#0D1B2A` — primary background
- **Ivory** `#F5F0E8` — light background, body text on dark
- **Gold** `#C9A84C` — accent, labels, headings
- **Teal** `#4A9B8E` — secondary accent
- **Display font**: Cormorant Garant (Google Fonts)
- **Body font**: Inter (Google Fonts)
